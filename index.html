Discord Quest Automation

Automate Discord quests with this JavaScript script

WARNING: Use at your own risk.

How to Use

  1. Open Discord in your browser (desktop version)
  2. Select the Quest you want to complete
  3. Open Developer Tools (F12 or Ctrl+Shift+I)
  4. Go to the Console tab
  5. Copy the code below and paste it into the console
  6. Press Enter to execute the script
Note: This script automates Discord quests by simulating game activity, video watching, and play activity. It works by intercepting Discord's Webpack modules.
